## ğŸŒ Konsep Dasar Internet of Things (IoT)

IoT didefinisikan sebagai jaringan global dinamis yang memungkinkan objek fisik dan virtual saling terhubung, berkomunikasi, serta bertukar data melalui protokol komunikasi standar.

Secara umum, sistem IoT terdiri dari beberapa **building blocks utama**:
1. **IoT Device** (sensor & aktuator)
2. **IoT Gateway**
3. **Network**
4. **Cloud / Backend**
5. **Application**

Pendekatan ini menekankan bahwa IoT bukan hanya perangkat, melainkan sebuah **sistem terintegrasi**.

---

## ğŸ§  IoT dan Embedded System

Embedded System merupakan inti dari perangkat IoT. Sistem ini berbasis mikroprosesor atau mikrokontroler dan dirancang untuk menjalankan fungsi tertentu secara real-time.

Dalam konteks IoT:
- Embedded system bertugas mengumpulkan data sensor
- Memproses data awal
- Mengirimkan data ke sistem cloud
- Berinteraksi dengan aplikasi pengguna

Peran embedded system sangat krusial dalam implementasi **Smart City**, seperti pada sistem transportasi cerdas, monitoring lingkungan, dan manajemen infrastruktur.
